# SoftForge HelpDesk

Repositório do projeto **SoftForge HelpDesk**, desenvolvido como parte do **Projeto Integrado Multidisciplinar (PIM IV)** do curso de **Análise e Desenvolvimento de Sistemas** da **UNIP**.

---

## Sobre o Projeto

O **SoftForge HelpDesk** é um sistema acadêmico para **gestão de chamados e suporte técnico**, desenvolvido para simular processos reais de atendimento em uma empresa de TI.

O sistema contempla registro de chamados, atribuição de técnicos, acompanhamento de prazos, histórico de ações, avaliação de atendimentos e comunicação interna entre solicitantes e técnicos.

Além disso, o projeto explora conceitos de **inteligência artificial** para auxiliar na classificação de chamados e sugestões de soluções, fornecendo um ambiente de aprendizado prático.
